# 📊 Superstore Sales, Profit & Orders Dashboard
Aspiring Data Analyst | Tableau | Excel | Power BI
This Tableau project presents an interactive analysis of Sales, Profit, and Orders using the Superstore dataset. The dashboard helps uncover business insights through dynamic visualizations, KPI tracking, and trend analysis.

## 🚀 Project Overview
The dashboard is designed to help stakeholders understand business performance, identify profitable areas, and make data-driven decisions.

## 🛠️ Tools Used
- Tableau Public
- Microsoft Excel

## 💡 Skills Demonstrated
- Data Visualization
- Dashboard Design
- KPI Development
- Business Analysis
- Data Storytelling
- Tableau

## 📈 Key Insights
- Technology category generated the highest sales.
- Consumer segment contributed the largest share of orders.
- Sales and profit showed positive growth trends.
- Regional analysis highlighted top-performing states.
